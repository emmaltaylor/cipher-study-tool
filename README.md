# Cipher Study Tool

A Python program to practice encryption using:

- Permutation Cipher
- Columnar Cipher
- Rail Fence Cipher

---

## Features

- Encrypt plaintext with three types of classical ciphers
- Handles plaintext of any length
- Pads incomplete blocks for clean output
- User-friendly CLI prompts
